QUESTION: Vilka typer av produkter kräver specialtillstånd för att säljas på Tradera?
ANSWER: Produkter som kräver specialtillstånd för att frakta, äga eller sälja behöver särskilda tillstånd för att säljas på Tradera. Detta kan vara anledningen till att din annons blev nedstängd om du inte hade tillståndet. 
URL: https://info.tradera.com/nedstangd-annons/

QUESTION: Varför kan min annons ha blivit nedstängd trots att jag inte har brutit mot någon regel?
ANSWER: Det kan hända att din annons inte var tillräckligt tydlig med att den följde reglerna eller att det fanns något som inte var i enlighet med företagets regler. Du kan alltid kontakta Tradera för att få mer information om varför din annons blev nedstängd.
URL: https://info.tradera.com/nedstangd-annons/

QUESTION: Hur gör jag för att undvika att min annons stängs ned?
ANSWER: Du kan läsa Traderas regelsida för att få mer information om företagets regler och vad som är acceptabelt att sälja på plattformen. På så sätt kan du undvika att bryta mot reglerna och få din annons nedstängd.
URL: https://info.tradera.com/nedstangd-annons/

QUESTION: Vilken typ av mail skickar Tradera när en annons bryter mot företagets regler?
ANSWER: När en annons bryter mot Traderas regler skickar företaget ett mail för att informera säljaren om vilken regel annonsen bröt mot. Mailet innehåller exempel på generella regelbrott, men anledningen till nedstängningen kan vara specifik för din annons.
URL: https://info.tradera.com/nedstangd-annons/

QUESTION: Hur kan jag hjälpa till att rapportera annonser som bryter mot reglerna?
ANSWER: Om du ser en annons som bryter mot Traderas regler kan du hjälpa till genom att rapportera annonsen till Tradera. på så sätt kan företaget snabbt titta på och ta bort annonsen om den bryter mot reglerna.
URL: https://info.tradera.com/nedstangd-annons/

QUESTION: Hur vet jag om min annons bryter mot Traderas regler?
ANSWER: Det enklaste sättet att undvika regelbrott är genom att läsa och följa Traderas regler för annonsering. Om du är osäker på om din annons följer reglerna eller inte kan du alltid kontakta företaget.
URL: https://info.tradera.com/nedstangd-annons/

QUESTION: Har Tradera rätt att stänga av mitt konto om jag bryter mot reglerna?
ANSWER: Ja, om du upprepar regelbrott trots flera varningar kan Tradera stänga av ditt konto. Detta sker dock vanligtvis bara om du fortsätter att göra samma fel gång på gång.
URL: https://info.tradera.com/nedstangd-annons/

QUESTION: Var kan jag hitta exempel på regler om vad som får säljas på Tradera?
ANSWER: Du kan hitta exempel på Traderas regler på företagets webbplats. Mer information om vad som är acceptabelt att sälja hittar du på regelsidan på Traderas webbplats.
URL: https://info.tradera.com/nedstangd-annons/

QUESTION: Vad händer om jag säljer alkohol på Tradera?
ANSWER: För att sälja alkohol på Tradera måste du uppfylla särskilda krav för att följa lagen och företagets regler. Om du säljer alkohol, se till att din annons tydligt beskriver att säljaren är 20 år eller äldre och att köparen också måste vara 20 år eller äldre.
URL: https://info.tradera.com/nedstangd-annons/

QUESTION: Kan jag lägga ut färdigpackade livsmedelsprodukter på Tradera?
ANSWER: Ja, du kan lägga ut färdigpackade livsmedelsprodukter på Tradera så länge de inte är utgångna och förpackningen är obruten. Varor som kan anses som hälsorisker eller som det är olagligt att sälja i butik, kan dock inte säljas på Tradera.
URL: https://info.tradera.com/nedstangd-annons/
