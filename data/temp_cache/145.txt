QUESTION: Vad innebär det negativa omdömet?
ANSWER: Det negativa omdömet beskriver en köpares eller säljares missnöje med en transaktion på Tradera.
URL: https://info.tradera.com/negativt-omdome/

QUESTION: Kan alla få ett negativt omdöme?
ANSWER: Ja, det kan hända alla köpare eller säljare på Tradera att få ett negativt omdöme.
URL: https://info.tradera.com/negativt-omdome/

QUESTION: Vad kan jag göra om jag har fått ett negativt omdöme?
ANSWER: När du fått ett negativt omdöme kan du kontakta säljaren eller köparen för att försöka lösa problemet eller lägga till en kommentar till ditt eget omdöme för att förklara din version av händelsen.
URL: https://info.tradera.com/negativt-omdome/

QUESTION: Kan jag ta bort ett negativt omdöme?
ANSWER: Nej, som säljare eller köpare kan du inte ta bort ett negativt omdöme, men du kan lägga till en kommentar till omdömet för att förklara din version av händelsen.
URL: https://info.tradera.com/negativt-omdome/

QUESTION: Hur påverkar ett negativt omdöme mina framtida transaktioner på Tradera?
ANSWER: Ett negativt omdöme kan påverka köparens eller säljarens anseende på Tradera och göra det svårare att genomföra framtida transaktioner.
URL: https://info.tradera.com/negativt-omdome/

QUESTION: Vad händer om jag inte kan lösa problemet med säljaren eller köparen?
ANSWER: Om du inte kan lösa problemet med säljaren eller köparen och det negativa omdömet fortfarande står kvar på ditt konto kan du kontakta Tradera för att få hjälp med att lösa konflikten.
URL: https://info.tradera.com/negativt-omdome/

QUESTION: Hur kan jag undvika negativa omdömen på Tradera?
ANSWER: För att undvika negativa omdömen på Tradera kan du vara tydlig med dina beskrivningar av produkterna, kommunicera tydligt med köpare eller säljare och skicka produkterna i tid och i beskrivet skick.
URL: https://info.tradera.com/negativt-omdome/

QUESTION: Kan jag rapportera en köpare eller säljare som gett mig ett negativt omdöme utan anledning?
ANSWER: Ja, du kan rapportera en köpare eller säljare till Tradera om de gett dig ett negativt omdöme utan anledning.
URL: https://info.tradera.com/negativt-omdome/

QUESTION: Hur länge står ett negativt omdöme kvar på mitt konto?
ANSWER: Ett negativt omdöme står kvar på ditt konto på Tradera i 12 månader. 
URL: https://info.tradera.com/negativt-omdome/

QUESTION: Vad händer om jag inte betalar för en vunnen auktion?
ANSWER: Om du inte betalar för en vunnen auktion kan säljaren ge dig ett negativt omdöme och Tradera kan blockera ditt konto från framtida köp. 
URL: https://info.tradera.com/negativt-omdome/
