QUESTION 1: Vad innebär det att samtycka till marknadsföring från Tradera och andra företag? 
ANSWER: Enligt texten på webbsidan kan Tradera sända marknadsföringsmaterial relaterade till Tradera-produkter och tjänster och produkter och tjänster från andra företag till användare som har godkänt att ta emot sådana meddelanden.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2018-2021/

QUESTION 2: Kan Tradera använda min personliga information för att anpassa min användarupplevelse och matcha mina intressen?
ANSWER: Ja, Tradera kan använda din personliga information till att skräddarsy vissa tjänster eller användarupplevelser för att bättre matcha dina intressen.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2018-2021/

QUESTION 3: Vilka personanpassade tjänster som erbjuds av Tradera på tredje parts webbplatser och onlinetjänster kan vara tillgängliga för mig?
ANSWER: Tradera kan tillhandahålla personanpassade tjänster som erbjuds av Tradera på tredje parts webbplatser och onlinetjänster. Dessa tjänster är speciellt utformade för att möta dina unika behov och intressen.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2018-2021/

QUESTION 4: Kan mina personuppgifter användas för att tillhandahålla anpassade annonser på tredje parts webbplatser?
ANSWER: Ja, Tradera kan använda dina personuppgifter för att tillhandahålla anpassade annonser, funktioner eller erbjudanden till dig på tredje parts webbplatser.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2018-2021/

QUESTION 5: Vilken typ av spårningsteknik används Tradera för att tillhandahålla online-tjänster?
ANSWER: Tradera kan använda cookies och andra spårningstekniker för att tillhandahålla onlinetjänster. De kan också arbeta med andra tredjepartsleverantörer såsom annons- eller analysföretag för att tillhandahålla dessa tjänster.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2018-2021/

QUESTION 6: Kan Tradera använda min platsinformation för att ge mig platsbaserade tjänster?
ANSWER: Ja, Tradera kan använda din platsinformation för att ge dig platsbaserade tjänster som annonsering, sökresultat och annat anpassat innehåll. 
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2018-2021/

QUESTION 7: Hur använder Tradera min information för att öka webbplatsernas och tjänsternas säkerhet?

URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2018-2021/

QUESTION 8: Kan Tradera använda min information för att hjälpa mig att hitta och komma i kontakt med andra personer?
ANSWER: Ja, Tradera kan använda din information för att göra det lättare för dig att hitta och få kontakt med andra personer på webbplatsen. 
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2018-2021/

QUESTION 9: Går Tradera vidare med min personliga information till andra företag utan mitt medgivande?
ANSWER: Enligt texten på webbsidan kommer Tradera inte att vidarebefordra användarnas personuppgifter till tredje parter utan användarens samtycke, förutsatt att de inte är skyldiga att göra det enligt lag.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2018-2021/

QUESTION 10: Kan jag välja att inte ta emot marknadsföringsmaterial eller ha min information delad med andra företag?
ANSWER: Ja, enligt texten på webbsidan kan användare välja att inte ta emot marknadsföringsmaterial eller ha sin information delad med andra företag genom att när som helst återkalla sitt samtycke.
