QUESTION: Vilka dokument behövs för att registrera ett företagskonto på Tradera?
ANSWER: För att Tradera ska kunna slutföra registreringen av ett företagskonto krävs det oftast underlag för att verifiera företagsuppgifterna. Som intyg kan man skicka in exempelvis utdrag ur Bolagsverket och företagets stadgar.
URL: https://info.tradera.com/underlag-foretagskonto/

QUESTION: Kan man använda sitt personliga konto för att sälja produkter på Tradera om man driver ett företag?
ANSWER: Nej, man kan inte använda sitt personliga konto på Tradera för att sälja produkter om man driver ett företag. Då behöver man registrera ett företagskonto och verifiera sina företagsuppgifter.
URL: https://info.tradera.com/underlag-foretagskonto/

QUESTION: Hur skickar man in underlag för att registrera ett företagskonto på Tradera?
ANSWER: För att skicka in underlag för att registrera ett företagskonto på Tradera kan man antingen scanna eller ta foton på de dokument som krävs och sedan ladda upp dem direkt på Traderas registreringssida för företagskonton.
URL: https://info.tradera.com/underlag-foretagskonto/

QUESTION: Hur lång tid tar det att få ett företagskonto godkänt på Tradera?
ANSWER: Det beror på hur snabbt man kan skicka in de krävda dokumenten och hur lång tid det tar för Tradera att verifiera dem. Normalt sett tar det ungefär 1-2 arbetsdagar att verifiera företagsuppgifterna.
URL: https://info.tradera.com/underlag-foretagskonto/

QUESTION: Kan man sälja varor utan att ha ett registrerat företagskonto på Tradera?
ANSWER: Ja, man kan sälja varor på Tradera utan att ha ett registrerat företagskonto. Men om man driver ett företag är det viktigt att se till att uppfylla de krav som ställs och att registrera ett företagskonto för att undvika problem i framtiden.
URL: https://info.tradera.com/underlag-foretagskonto/

QUESTION: Måste man vara momsregistrerad för att sälja på Tradera?
ANSWER: Ja, om man vill sälja varor på Tradera och tjäna pengar måste man vara momsregistrerad. Som företagare kan man ansöka om att bli momsregistrerad genom Skatteverket.

QUESTION: Kan man göra avdrag på momsen om man säljer varor på Tradera som företag?
ANSWER: Ja, som momsregistrerat företag kan man göra avdrag på momsen när man säljer varor på Tradera. Man kan då dra av momsen på kostnaderna för att köpa in varorna.

QUESTION: Kan man registrera flera företagskonton på Tradera?
ANSWER: Ja, man kan registrera flera företagskonton på Tradera om man driver flera företag. Varje företagskonto måste dock verifieras separat.
