QUESTION: Vad är Bank ID?
ANSWER: Bank ID är en e-legitimation som du använder för att styrka din identitet på Internet, exempelvis på banken, hos Försäkringskassan, och nu även på Tradera. Bank ID är Sveriges i särklass största e-legitimation och utfärdas av 12 banker och blir allt mer vanlig. 
URL: https://info.tradera.com/sakerhetscenter/fragor-och-svar-bank-id/

QUESTION: Vad händer när jag verifierar mig med Bank ID hos Tradera?
ANSWER: När du verifierar dig med Bank ID hos Tradera klickar du upp din applikation på din dator, telefon eller surfplatta och fyller i din sexsiffriga kod. Därefter skickas din identitet säkert till oss och verifieras mot den information vi har om dig så att vi kan styrka att det verkligen är du som loggar in.
URL: https://info.tradera.com/sakerhetscenter/fragor-och-svar-bank-id/

QUESTION: Är Bank ID säkert att använda på Tradera?
ANSWER: Ja, Bank ID är ett säkert sätt för dig att verifiera din identitet på Tradera.

QUESTION: Kan jag använda Bank ID på min mobil eller surfplatta?
ANSWER: Ja, du kan använda Bank ID på din mobil eller surfplatta för att verifiera din identitet på Tradera.

QUESTION: Vad händer om jag inte har Bank ID?
ANSWER: Om du inte har Bank ID kan du inte verifiera din identitet på Tradera och delta i auktioner eller handla. Det går dock att skaffa Bank ID hos din bank om du inte redan har det.
URL: https://info.tradera.com/sakerhetscenter/fragor-och-svar-bank-id/

QUESTION: Är Bank ID obligatoriskt att använda på Tradera?
ANSWER: Nej, det är frivilligt att använda Bank ID på Tradera. Du kan också använda dig av SMS-koder för att verifiera din identitet.
