QUESTION: Vilka kontaktsätt kan jag använda för att nå Traderas kundtjänst?
ANSWER: Du kan nå Traderas kundtjänst genom att kontakta dem via kontaktformuläret, telefon eller e-post. Deras kontaktuppgifter finns på Traderas hemsida.
URL: https://info.tradera.com/en/dbschenker/

QUESTION: Hur kan jag skapa ett konto på Tradera?
ANSWER: För att skapa ett konto på Tradera behöver du gå till Traderas hemsida och klicka på "Bli medlem". Därefter behöver du fylla i dina personuppgifter och välja ett användarnamn och lösenord.
URL: https://info.tradera.com/en/dbschenker/

QUESTION: Kan jag sälja saker på Tradera som privatperson?
ANSWER: Ja, som privatperson kan du sälja på Tradera. Däremot finns det olika regler och krav som du behöver följa för att få sälja som privatperson på plattformen. Tradera ger omfattande information om detta på sin hemsida.
URL: https://info.tradera.com/en/dbschenker/

QUESTION: Hur fungerar Traderas fraktberäknare?
ANSWER: Traderas fraktberäknare används för att beräkna fraktkostnaden för en vara som du köper eller säljer på plattformen. Beräkningen baseras på varans vikt och dimensioner och varifrån och vart den ska skickas. Du kan hitta mer information om Traderas fraktberäknare på deras hemsida.
URL: https://info.tradera.com/en/dbschenker/

QUESTION: Vilka betalningsmetoder accepterar Tradera?
ANSWER: På Tradera kan man betala med kreditkort, PayPal eller via direkt banköverföring. Detta anges i samband med köpet.
URL: https://info.tradera.com/en/dbschenker/

QUESTION: Kan jag starta en webbutik på Tradera?
ANSWER: Ja, det går att starta en webbutik på Tradera. För att göra detta behöver du sälja regelbundet på plattformen och uppfylla vissa krav. Mer information om webbutiker på Tradera finns på deras hemsida.
URL: https://info.tradera.com/en/dbschenker/

QUESTION: Hur fungerar Traderas returpolicy?
ANSWER: Tradera tillåter säljare att ange sina egna returvillkor. Dock finns det vissa minimikrav som måste uppfyllas för att köparna ska ha rätt att returnera en vara. Information om Traderas returpolicy finns tillgänglig på deras hemsida.
URL: https://info.tradera.com/en/dbschenker/

QUESTION: Vad är charities på Tradera?
ANSWER: Charities på Tradera är organisationer som har registrerat sig på plattformen för att försöka samla in pengar för välgörenhetsändamål. Säljare kan välja att donera en del av sin försäljning till en organisation och köpare kan också donera direkt till organisationskonton som finns på Tradera.
URL: https://info.tradera.com/en/dbschenker/

QUESTION: Vilka populära varumärken finns på Tradera?
ANSWER: Det finns många populära varumärken som säljs på Tradera, bland andra Acne Studios, Marimekko och Gant. För att se olika kategorier och märken kan du gå in på Traderas hemsida.
URL: https://info.tradera.com/en/dbschenker/

QUESTION: Hur kan jag sälja på Tradera som företag?
ANSWER: För att sälja på Tradera som företag krävs att man registrerar sig som en företagssäljare på plattformen. Mer information om att sälja som företag på Tradera finns tillgänglig på deras hemsida.
URL: https://info.tradera.com/en/dbschenker/
