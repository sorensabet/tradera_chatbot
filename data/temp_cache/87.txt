QUESTION: Vilka åtgärder kan vidtas om en användare orsakar problem eller risker för skadeståndsskyldighet enligt Användaravtalet?
ANSWER: Om en användare orsakar problem eller risker för skadeståndsskyldighet i enlighet med Användaravtalet kan Tradera vidta åtgärder som att ta bort vissa statusuppdateringar som hör till kontot, ta bort eller minska rabatter och använda tekniska och rättsliga åtgärder för att hindra användaren från att använda Traderas tjänster. Dessa åtgärder kan vidtas om Tradera anser att sådana begränsningar kommer att öka säkerheten för Traderas användare eller minska risken för att Tradera eller Traderas användare ådrar sig betalningsskyldighet, om användaren gör intrång i tredje parts rättigheter, om användaren agerar i strid med ändamålet med detta Användaravtal eller Traderas policys eller kränker Traderas anställda eller användare, om Tradera inte kan bekräfta den information som tillhandahållits av användaren eller om användaren inte betalar avgifterna för Traderas tjänster vid fallen betalningsdag.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Vilka andra påföljder kan Tradera tillgripa om användare bryter mot Användaravtalet?
ANSWER: Denna bestämmelse i Användaravtalet begränsar inte andra påföljder som Tradera har rätt till, så utöver de tidigare nämnda åtgärderna kan Tradera tillgripa andra påföljder mot användare som bryter mot Användaravtalet om Tradera anser detta lämpligt.
