QUESTION 1: Vilket syfte har denna Integritetspolicy?
ANSWER: Denna Integritetspolicy är avsedd att förklara hur Tradera samlar in, använder och skyddar din personliga information.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/

QUESTION 2: Varför samlar Tradera in personlig information?
ANSWER: Tradera samlar in personlig information för att förse dig med de tjänster och den kundservice som du begär. Detta gör att de kan förhindra, upptäcka och utreda bedrägerier, hantera tvister och utveckla och förbättra Traderas tjänster.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/

QUESTION 3: Är alla aktiviteter tillåtna enligt Traderas Användaravtal?
ANSWER: Nej, Tradera förhindrar potentiellt förbjudna eller olagliga aktiviteter som inte följer deras användaravtal.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/

QUESTION 4: Vad gör Tradera om de upptäcker bedrägeri eller säkerhetsincidenter?
ANSWER: Om Tradera upptäcker bedrägeri eller säkerhetsincidenter, kommer de att förhindra, utreda och lösa problemet så snabbt som möjligt för att skydda dig och andra användare.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/

QUESTION 5: Använder Tradera min personliga information för direktreklam?
ANSWER: Ja, Tradera kan använda din personliga information för direktreklam och andra reklamerbjudanden i enlighet med tillämpliga regler och dina meddelandepreferenser.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/

QUESTION 6: Kan Tradera kontakta mig via telefon?
ANSWER: Ja, Tradera kan kontakta dig via telefon för att lösa tvister, driva in avgifter och lösa problem med ditt konto eller Traderas webbplatser, tjänster, applikationer eller verktyg.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/

QUESTION 7: Använder Tradera automatisk uppringningsmetod eller förinspelade meddelanden för att kontakta mig?
ANSWER: Ja, Tradera kan använda automatisk uppringningsmetod eller förinspelade meddelanden för att kontakta dig på något telefonnummer, i enlighet med tillämpliga regler och för de ändamål som beskrivs i denna Integritetspolicy.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/

QUESTION 8: Hur använder Tradera min personliga information för att förbättra sina tjänster?
ANSWER: Tradera använder din personliga information för att anpassa, mäta och förbättra deras tjänster, innehåll och annonser. På så sätt kan de ge dig en bättre användarupplevelse.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/

QUESTION 9: Använder Tradera push-notifieringar?
ANSWER: Ja, Tradera kan skicka push-notifieringar med anledning av din budgivning eller annan användning av deras appar och tillhandahålla dig andra tjänster som du begär.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/

QUESTION 10: Vilka rättigheter har jag över min personliga information hos Tradera?
ANSWER: Du har rätt att få tillgång till, ändra eller radera din personliga information från Traderas databaser. Du kan också välja att inte få direktreklam från Tradera. Mer information finns i deras Integritetspolicy.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2014-2015/
