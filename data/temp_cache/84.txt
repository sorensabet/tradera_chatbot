QUESTION: Vilka åtgärder kan leda till att jag stängs av från Tjänsterna på Tradera?
ANSWER: Åtgärder som kan leda till att du stängs av från Tjänsterna på Tradera inkluderar att manipulera föremåls pris eller andra användares annonser, publicera eller använda falskt, felaktigt, vilseledande eller kränkande innehåll eller uppgifter, skicka eller publicera skräppost eller använda automatiserade verktyg eller programvara. En fullständig lista av ageranden som kan leda till att du stängs av finns i användaravtalet.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Vad händer om jag inte betalar för ett föremål jag köpt på Tradera?
ANSWER: Om du inte betalar för ett föremål som du har köpt på Tradera kan du stängas av från Tjänsterna. Detta gäller inte om säljaren har ändrat objektbeskrivningen väsentligt efter ditt bud, om ett tydligt skrivfel har gjorts eller om du inte kan kontakta säljaren. Det är alltid viktigt att följa reglerna för köpare på Tradera.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Kan jag använda Traderas Tjänster om jag är under 18 år gammal?
ANSWER: Du kan inte använda Traderas Tjänster om du inte kan ingå bindande avtal, t.ex. om du är under 18 år gammal. Detta anges i användaravtalet på Tradera.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Vad kan hända om jag överlåter mitt Traderakonto till någon annan utan samtycke från Tradera?
ANSWER: Om du överlåter ditt Traderakonto och användar-ID till någon annan utan samtycke från Tradera, kan det leda till att ditt konto stängs av. Du måste alltid be om och få samtycke från Tradera innan du överlåter ditt konto till någon annan person.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Vad kan jag göra om jag inte kan kontakta säljaren eller köparen efter ett köp eller försäljning på Tradera?
ANSWER: Om du inte kan kontakta säljaren eller köparen efter ett köp eller försäljning på Tradera, finns det regler för köpare och säljare som du kan följa. Om du inte kan kontakt med säljaren kan du begära en återbetalning från Tradera och om du inte kan kontakta köparen kan du kontakta Tradera och begära hjälp med att få betalning för försäljningen.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Går det att använda automatiserade verktyg eller programvara på Tradera?
ANSWER: Det går inte att använda automatiserade verktyg eller programvara på Tradera. Om du gör det kan det leda till att du stängs av från Tjänsterna. Detta finns specificerat i användaravtalet på Tradera.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Kan jag publicera falskt eller vilseledande innehåll på Tradera?
ANSWER: Nej, du får inte publicera falskt, felaktigt, vilseledande eller kränkande innehåll eller uppgifter på Tradera. Om du gör det kan det leda till att du stängs av från Tjänsterna. Du kan läsa mer om Traderas policys kring omdömen och riktlinjer kring uppförande i användaravtalet.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Kan jag använda en annan persons användar-ID eller lösenord på Tradera?
ANSWER: Nej, du får inte använda en annan persons användar-ID eller lösenord på Tradera. Detta kan leda till att ditt konto stängs av. Användaravtalet på Tradera förbjuder detta.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Vilka typer av meddelanden är förbjudna på Tradera?
ANSWER: Följande typer av meddelanden är förbjudna på Tradera: skräppost, oönskade meddelanden, massmeddelanden, kedjebrev eller pyramidspel. Om du skickar dessa typer av meddelanden kan det leda till att du stängs av från Tjänsterna på Tradera.
URL: https://info.tradera.com/anvandaravtal-2019-2020/

QUESTION: Är det tillåtet att distribuera virus eller annan teknik som kan skada Tradera på plattformen?
ANSWER: Nej, det är inte tillåtet att distribuera virus eller annan teknik som kan skada Tradera eller användarnas intressen eller tillgångar. Om du gör detta kan det leda till att du stängs av från Tjänsterna. Detta specifieras i användaravtalet på Tradera.
URL: https://info.tradera.com/anvandaravtal-2019-2020/
