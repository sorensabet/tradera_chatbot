QUESTION: Varför samlar Tradera in personuppgifter?
ANSWER: Tradera samlar in personuppgifter för att ge användarna tillgång till webbplatsen och dess tjänster, för att förhindra bedrägerier, för att anpassa och förbättra tjänsterna och för att kunna kontakta användarna vid behov.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2021-2023/

QUESTION: Vad gör Tradera med informationen som samlas in?
ANSWER: Tradera använder informationen för att anpassa, mäta och förbättra sina tjänster, kontakta användare vid behov, och för att skicka marknadsföringsmaterial om Tradera-produkter och onlinetjänster, samt produkter och tjänster från andra företag. De kan också använda informationen för att skräddarsy vissa tjänster eller användarupplevelser för att bättre matcha användarens intressen.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2021-2023/

QUESTION: Vad gör Tradera med användarnas personuppgifter utan deras samtycke?
ANSWER: Tradera använder användarnas personuppgifter utan deras samtycke för att ge dem tillgång till webbplatsen och dess tjänster, för att förhindra bedrägerier, för att kontakta användare vid behov, och för att upprätthålla Användaravtalet.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2021-2023/

QUESTION: Vilka åtgärder vidtar Tradera för att skydda användarnas personuppgifter?
ANSWER: Tradera vidtar tekniska och organisatoriska åtgärder för att skydda användarnas personuppgifter från förlust, missbruk, obehörig åtkomst eller spridning, såsom kryptering och säkerhetskopiering.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2021-2023/

QUESTION: Kan användare begära att få tillgång till sina personuppgifter som Tradera har samlat in?
ANSWER: Ja, användare har rätt att begära att få tillgång till sina personuppgifter som Tradera har samlat in. De kan också begära att uppgifterna ska rättas, raderas eller begränsas i behandlingen.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2021-2023/

QUESTION: Vilka är de juridiska grunderna för Traderas behandling av personuppgifter?
ANSWER: Tradera behandlar personuppgifter på grundval av samtycke, kontraktsenlig behandling, rättslig förpliktelse, ett berättigat intresse eller som en uppgift av allmänt intresse.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2021-2023/

QUESTION: Kan användare välja att inte ta emot marknadsföringsmaterial från Tradera?
ANSWER: Ja, användare kan välja att inte ta emot marknadsföringsmaterial från Tradera genom att avregistrera sig från utskicken eller kontakta Tradera direkt.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2021-2023/

QUESTION: Kan personuppgifter delas med tredje part?
ANSWER: Ja, Tradera kan dela personuppgifter med tredje part för att uppfylla de ändamål som beskrivs i integritetspolicyn, men endast om de har tillräckliga skyddsåtgärder i plats för att skydda användarnas personuppgifter.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2021-2023/

QUESTION: Hur länge lagrar Tradera personuppgifter?
ANSWER: Tradera lagrar personuppgifter så länge det är nödvändigt för att uppfylla syftet med insamlingen av uppgifterna, eller så länge det krävs enligt lag.
URL: https://info.tradera.com/sakerhetscenter/integritetspolicy-2021-2023/

QUESTION: Kan användare kräva att Tradera raderar sina personuppgifter?
ANSWER: Ja, användare kan begära att Tradera raderar deras personuppgifter, förutsatt att det inte finns något lagstadgat krav på att behålla uppgifterna eller att Tradera har ett berättigat intresse av att behålla dem.
