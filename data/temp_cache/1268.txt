QUESTION: Vad händer om jag bryter mot användarvillkoren på Tradera?
ANSWER: Om du bryter mot användarvillkoren på Tradera kan du förvägra dig att sälja fler varor. Tradera kan också få tillgång till information från dig som säljare för att kunna bedöma om du bryter mot villkoren.
URL: https://info.tradera.com/anvandaravtal-2022-2023/

QUESTION: Kan Tradera begränsa min användning av Tjänsterna?
ANSWER: Ja, Tradera kan besluta att helt eller delvis begränsa en företagsanvändares användning av Tjänsterna och kommer att meddela den berörda företagsanvändaren senast när beslutet träder i kraft.
URL: https://info.tradera.com/anvandaravtal-2022-2023/

QUESTION: Kan jag sälja varor som står i strid med Traderas etiska policy?
ANSWER: Nej, du kan inte sälja varor eller tjänster som står i strid med Traderas vid var tid gällande etiska policy för respektive land, eller annars agera på ett sätt som rimligen kan anses stå i konflikt med vedertagna regler från kortföretagen eller från deras branschorganisation eller några andra betaltjänstvillkor som är relevanta för Tradera.
URL: https://info.tradera.com/anvandaravtal-2022-2023/

QUESTION: Kan jag använda Traderas betallösning?
ANSWER: Ja, du kan använda Traderas betallösning. Om du använder betallösningen behöver du också följa villkoren för den.
URL: https://info.tradera.com/anvandaravtal-2022-2023/

QUESTION: Vad händer om jag ger felaktig eller missledande information till Tradera?
ANSWER: Om du ger felaktig eller missledande information till Tradera eller döljer relevanta omständigheter kan du förvägra dig att sälja fler varor på Tradera.
URL: https://info.tradera.com/anvandaravtal-2022-2023/

QUESTION: Vad är företagsanvändares användning av Tjänsterna?
ANSWER: Företagsanvändares användning av Tjänsterna innebär användning av Tradera som ett företag, till exempel för försäljning av produkter i stora mängder på regelbunden basis eller med andra affärsmässiga syften.
URL: https://info.tradera.com/anvandaravtal-2022-2023/

QUESTION: Finns det en etisk policy för Tradera?
ANSWER: Ja, det finns en etisk policy för Tradera som styr vad som är tillåtet att sälja på plattformen.
URL: https://info.tradera.com/anvandaravtal-2022-2023/

QUESTION: Vad händer om jag agerar på ett sätt som kan anses stå i konflikt med vedertagna regler från kortföretagen eller från deras branschorganisation eller några andra betaltjänstvillkor som är relevanta för Tradera?
ANSWER: Om du agerar på ett sätt som rimligen, direkt eller indirekt, kan anses stå i konflikt med vedertagna regler från kortföretagen eller från deras branschorganisation eller några andra betaltjänstvillkor som är relevanta för Tradera, kan du förvägra dig att sälja fler varor på Tradera.
URL: https://info.tradera.com/anvandaravtal-2022-2023/

QUESTION: Kan jag sälja varor på Tradera om jag är på obestånd?
ANSWER: Nej, om du kan antas vara på obestånd kan du förvägra dig att sälja fler varor på Tradera.
URL: https://info.tradera.com/anvandaravtal-2022-2023/

QUESTION: Vad är Villkoren för Traderas betallösning?
ANSWER: Villkoren för Traderas betallösning är regler som du behöver följa om du väljer att använda Traderas betallösning.
