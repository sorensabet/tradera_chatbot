QUESTION: Vad är viktigt att tänka på vid köp av produkter på Tradera?
ANSWER: När du köper produkter på Tradera är det viktigt att du noga utvärderar objektet innan du lägger din beställning. Det innebär att du kan behöva ta reda på information som storlekar, upplaga eller funktionalitet på en produkt. Du bör också kolla upp totalkostnaden för frakt och veta vilka betalningsalternativ som säljaren accepterar. Du kan alltid kontakta säljaren om något är oklart.
URL: https://info.tradera.com/sakerhetscenter/kopa-sakert/utvardera-objektet/

QUESTION: Vad kan jag göra om jag känner mig osäker på en affär på Tradera?
ANSWER: Om du känner dig osäker på en affär på Tradera är det viktigt att du litar på dina instinkter och ser över affären igen. Om något känns misstänksamt, så bör du kontakta säljaren innan du betalar, gärna över telefon. Du har också alltid möjlighet att rapportera misstänkta medlemmar till Tradera via funktionen ”Anmäl till granskning” som du hittar på respektive annons.
URL: https://info.tradera.com/sakerhetscenter/kopa-sakert/utvardera-objektet/

QUESTION: Var hittar jag information om hur jag handlar på Tradera?
ANSWER: Du kan hitta information om hur du handlar på Tradera genom att besöka handelsplatsens webbsida. Klicka på "Säkerhet" och sedan på "Vanliga frågor och svar". Här kan du hitta svar på frågor om handel på Tradera.
URL: https://info.tradera.com/sakerhetscenter/kopa-sakert/

QUESTION: Hur kan jag söka efter produkter på Tradera?
ANSWER: Du kan söka efter produkter på Tradera genom att använda sökfältet på handelsplatsens webbsida. Skriv in namnet på den produkt du söker och välj sedan det omslag som matchar din sökning. Alternativt kan du navigera på handelsplatsen genom att välja en kategori, ett varumärke eller annat som kan hjälpa dig att hitta det du söker efter.
URL: https://www.tradera.com/

QUESTION: Kan jag sälja produkter på Tradera som företag?
ANSWER: Ja, du kan sälja produkter på Tradera som företag genom att skapa ett företagskonto på handelsplatsens webbsida. Frågor som rör försäljning av produkter genom företagskonto kan besvaras genom att besöka handelsplatsen.
URL: https://www.tradera.com/

QUESTION: Vad innebär Grön frakt på Tradera?
ANSWER: Grön frakt är en frakttjänst som Tradera erbjuder för att motverka klimatförändringar. Genom Grön frakt så kan säljare och köpare köpa tjänsten till ett reducerat pris där Tradera sedan köper klimatkompenserande projekt som gör att köparen kompenserar för den miljöpåverkan som frakten har gett upphov till.
URL: https://www.tradera.com/

QUESTION: Finns det en fraktkalkylator på Tradera?
ANSWER: Ja, det finns en fraktkalkylator på Tradera som du kan använda för att räkna ut fraktkostnaden på en produkt. Du hittar länken till fraktkalkylatorn under fliken "Frakt och betalning" på produktsidan.
URL: https://info.tradera.com/sakerhetscenter/kopa-sakert/utvardera-objektet/

QUESTION: Vad är Vanliga frågor och svar på Tradera?
ANSWER: Vanliga frågor och svar på Tradera är en sida som innehåller frågor och svar om handel på handelsplatsen. Här kan du hitta svar på de vanligaste frågorna angående Tradera.
URL: https://info.tradera.com/sakerhetscenter/kopa-sakert/

QUESTION: Kan jag donera med Tradera?
ANSWER: Ja, du kan donera med Tradera genom att skänka pengar till välgörenhetsorganisationer genom handelsplatsens webbsida. Du hittar mer information om hur detta fungerar genom att besöka handelsplatsen.
URL: https://www.tradera.com/

QUESTION: Hur kan jag kontakta Tradera?
ANSWER: Du kan kontakta Tradera genom att besöka handelsplatsens webbsida och klicka på "Kontakt & hjälp". Här finns information om hur du kan ta kontakt med Traderas kundtjänst genom telefon, mejl eller chatt.
URL: https://www.tradera.com/
